# C++ 与 计算机视觉学习沉淀

本项目用于记录大一期间自学 C/C++ 编程以及 OpenCV 计算机视觉基础的实践成果。

## 📁 项目目录说明

### 1. C语言-学生信息管理系统
* **简介**：大一上学期独立完成的实践项目。
* **核心技术**：C语言链表（Linked List）数据结构、结构体应用、本地文件读写（实现数据持久化存储）。
* **功能**：支持学生信息的增删改查及保存。

### 2. OpenCV-证件照换底
* **核心技术**：基于 Qt 与 OpenCV 实现。应用了 HSV 颜色空间转换、阈值分割、形态学操作以及图像掩膜（Mask）合成。
* **功能**：自动识别特定背景颜色的证件照，实现精准抠图与背景颜色替换。

### 3. OpenCV-红布隐身术
* **核心技术**：颜色区域识别、像素级逻辑运算（AND/OR/NOT）、纯背景替换。
* **功能**：静态图像中的特定颜色（红布）消融，达到趣味隐身效果。
